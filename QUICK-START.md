## 🚀 BarberShop - Início Rápido

### ✅ Você descobriu que não tem MongoDB?

**Ótimo! Siga uma destas opções:**

---

### 📖 Opção A: Usar MongoDB na Nuvem (Recomendado - SEM INSTALAR)

```
1. Leia: SETUP-ATLAS.md
2. Leva 5 minutos
3. Sem instalar nada
```

👉 **[Abra SETUP-ATLAS.md](SETUP-ATLAS.md)**

---

### 📦 Opção B: Instalar MongoDB Local

#### Windows (Automático)
```powershell
.\install-mongodb.ps1
```

#### Windows (Manual)
Baixe: https://www.mongodb.com/try/download/community

#### macOS
```bash
brew install mongodb-community
brew services start mongodb-community
```

---

### ▶️ Depois escolher MongoDB, execute:

**Terminal 1:**
```bash
# Se MongoDB Local
mongod

# Se MongoDB Atlas, pule este passo
```

**Terminal 2:**
```bash
cd backend
npm run dev
```

**Terminal 3:**
```bash
cd frontend
npm start
```

✔️ Acesse: http://localhost:3000
